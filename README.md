# business-website-project


○ Project description 
 Coffee Blends is a responsive coffee shop and restaurant website that showcases the brand’s story, menu, and products while supporting basic e-commerce features like product listing, cart, and checkout pages. project includes dedicated sections for online ordering, table booking, blog posts, and customer testimonials to provide a complete digital experience for a modern café business.

○ Tech used 
HTML+CSS+JavaScript
